# FM Modulation & Waveform Display Platform – STM32F103 + RT0803L

本项目是一套基于 **STM32F103C8T6** 的 FM 调制与波形显示学习平台。  
系统能够采集模拟前端信号、进行调理与频率测量，并通过 **RT0803L** 完成 FM 信号发射，同时在 **1.8 寸 TFT 屏** 上实时显示波形。

> 📌 **项目参考**：本项目的波形显示与基础框架参考了 [简易数字示波器设计（入门版）](https://oshwhub.com/course-examples/yi-qi-yi-biao-jian-yi-shu-zi-shi-bo-qi-she-ji-cha-jian-ban)，在此基础上对模拟前端、频率采集方案及信号发射部分进行了独立设计与优化。

---

## ✨ 功能特性

- **模拟信号实时采样与波形显示**
- **频率测量（基于 TLV3702高速比较器）**
- **FM 调制度调节**
- **RT0803L FM 发射功能**
- **可切换 ×1 / ×50 衰减档位**
- **Type-C 供电接口 + 同轴输入接口**

---

## 🛠 硬件设计

硬件主要模块：

### **1）模拟前端**
- 运放：**TL072**（信号调理、增益切换）
- 比较器：**TLV3702**（轨对轨高速比较器）
- 衰减器：**AD5112 数字电位器**

改进点：

- 将原开源项目使用的 **LM393** 替换为 **TLV3702**，大幅提升高频触发准确性。
- 微调模拟链路，确保无论 AC 或 DC 输入，都能保持在 STM32 ADC 允许范围内。

### **2）MCU 与控制**
- **STM32F103C8T6**（可替换 **GD32E230C8T6**）
- I/O 用于采样控制、频率测量、驱动屏幕、控制数字电位器

### **3）FM 发射**
- 发射芯片：**RT0803L**
- 通过 MCU 调节AD511调整输入电压来调制度与载波参数

### **4）接口与供电**
- Type-C 供电
- 同轴信号输入接口（降低噪声、提高测试稳定性）

硬件目录：`/hardware/`

## 📌 设计亮点

### ✔ 1. 解决了开源方案的核心问题：频率触发不准  
开源示波器的 LM393 因无法触发接近轨压的高频信号 → 导致频率测量偏差。  
本项目替换为 **TLV3702ID** 后，触发点可覆盖几乎全范围，提高准确性。

### ✔ 2. 完整调理链路自主搭建  
运放 + 衰减网络 + 比较器  
保证信号始终落在可处理范围内，显示波形更稳定。

### ✔ 3. FM 发射端由本人独立编写  
RT0803L 的寄存器配置、调制度控制由本人手写驱动完成。
